## ✨ La Estética y Misión

**MTB Gear** fue diseñado pensando en el usuario aventurero, dinámico y que busca la máxima calidad. Para reflejar estos valores, el proyecto adopta una estética moderna de *arquitectura clara (Light Theme)* estructurada con una paleta vibrante:

- 🏔️ **Fondos Light & Minimalistas:** Un entorno espacioso y pulcro (`#f8fafc`) que sirve de lienzo perfecto para destacar imágenes de alta calidad (como la MTB naranja protagonista).
- 🔥 **Acentos Naranja Vibrante (`#ff6a00`):** Utilizados para destacar los elementos de interfaz principales (botones de acción, el logo principal, métricas clave), inspirando adrenalina y la energía del ciclismo extremo.
- 💨 **Tipografía Moderna (Outfit):** Empleada por su excelente legibilidad en pantallas digitales, su geometría equilibrada y ese aire dinámico y técnico que necesita una tienda deportiva premium.
- 🎨 **Animaciones Fluidas & Glassmorphism:** Una barra de navegación translúcida para generar interactividad y profundidad, sombras activas en los botones de "Call to Action", elementos visuales rotatorios (el gráfico de fondo) y montajes de transición de entrada para dar al usuario la sensación palpable de que está ante un portal de máximo nivel.

## 🚀 Instalación y Uso de Desarrollo

Para explorar este proyecto en tu propio entorno:

1. **Clonar este repositorio:**
```bash
git clone https://github.com/javiboin/ecommerce-mtb-react.git
```

2. **Navegar al directorio del proyecto:**
```bash
cd ecommerce-mtb-react
```

3. **Instalar dependencias:**
```bash
npm install
```

4. **Correr el servidor de desarrollo manejado por Vite:**
```bash
npm run dev
```

## 🛠️ Estructura Principal Implementada

- **`src/App.jsx`**: Aloja la estructura integral de la **Hero Section**, englobando componentes semánticos (Navbar envolvente, panel de texto con "Call to Action" e indicadores, y área visual adaptativa de la montaña).
- **`src/index.css`**: Define las variables maestras (`:root`), configuraciones globales y todas las interacciones dinámicas animadas mediante *keyframes* nativos (`fadeIn`, `float`, `slideUp`) construyéndose sobre Vanilla CSS puro en su máxima expresión de rendimiento.
